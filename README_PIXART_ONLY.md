# FP4DiT PixArt-α Minimal Code Split

This directory is a code-only subset copied from the original repo to run FP4DiT for PixArt-α.

Included code:
- `scripts/pixart_alpha_calib.py`
- `scripts/pixart_alpha_brecq.py`
- `qdiff/`
- `ldm/`
- `ddim/`
- `utils/`
- `environment.yml`, `setup.py`, `LICENSE`, `README.md`

Not copied:
- unrelated model directories and configs not needed by PixArt-α flow
- dataset/caption/calibration artifacts

External runtime artifacts you still need to provide:
- calibration output file: `pixart_calib_brecq.pt` (generated by `scripts/pixart_alpha_calib.py`)
- prompt/caption assets if you use caption modes (`--coco_10k`, `--coco_9k`, `--pixart`, `--coco2014`, `--hpsv2`), under `captions/`
- model downloads from Hugging Face (e.g. PixArt-XL and safety checker)

If you want to avoid caption files, pass a direct prompt with:
- `python scripts/pixart_alpha_brecq.py --prompt "..." ...`
